# Pages 目录概述

## square

- **index.wxml**: 定义广场页面的结构，包括横幅、快捷操作和列表区域。用户可以在此页面查看和参与目标招募活动。
- **index.js**: 包含广场页面的逻辑处理，包括数据加载、事件处理等。
- **index.wxss**: 定义广场页面的样式，使页面元素具有一致的外观和布局。
